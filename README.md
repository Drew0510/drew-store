# Drew Store Profile Website

## Cách chạy
1. Giải nén file.
2. Mở file `index.html` bằng trình duyệt.
3. Để đưa lên mạng miễn phí, có thể dùng GitHub Pages, Netlify hoặc Vercel.

## Những chỗ cần sửa
- Link Discord: tìm `https://discord.com/`
- Link Facebook, TikTok ở phần footer
- Tên và số liệu khách hàng
- Nội dung dịch vụ
- Màu chính trong phần `:root`
